# BuildIntel Sample Assets

This folder contains lightweight, synthetic sample documents that are safe to keep in the repository.

Included files:
- `residential-floor-plan-sample.pdf`
- `commercial-mep-plan-sample.pdf`

These assets are intentionally:
- small enough for normal git usage
- sanitized and demo-friendly
- suitable for screenshots, walkthroughs, and lightweight document-flow testing

The larger source plan files used during local experimentation stay outside the tracked sample pack so the repository remains fast to clone and easier to maintain.
