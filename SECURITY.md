# Security Policy

## Supported Versions

This project is under active development. Security fixes are applied to the latest code on the `main` branch.

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

To report a vulnerability:

1. Email `carldavenquimoyog@gmail.com` with the subject line `BuildIntel Security Report`.
2. Include a clear description of the issue, affected area, impact, and steps to reproduce.
3. If available, include screenshots, logs, proof of concept details, and suggested mitigations.

You can expect an acknowledgment within 3 business days. Valid reports will be reviewed privately, triaged based on severity, and patched as quickly as practical.
